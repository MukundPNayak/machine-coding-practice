🎯 Problem Statement: React Machine Coding

Build an interactive Kanban Board (like Trello) in React.

Core Requirements

Columns:

At least 3 columns (e.g., Backlog, In Progress, Done).

Tasks:

Each column contains a list of cards (tasks).

Each card has:

Title

Description

Unique ID

Drag & Drop:

You should be able to drag a card from one column to another.

CRUD on Cards:

Add a new card to any column.

Edit card title/description in place.

Delete a card.

Persistent State:

Use localStorage to save board state so refresh keeps changes.

✨ Bonus Features (not required but valuable)

Reorder cards within a column via drag.

Column add/remove.

Animate drag & drop.

Input validation & user notifications.

🛠 Technical Constraints

✔ Use React (functional components + hooks)
✔ You can use open-source drag-and-drop library (e.g., react-beautiful-dnd)
✔ No backend required — persistence via localStorage
✔ Code should be modular, maintainable, and well-structured
